Receive a daily email if you have any plugins that require updating.

== Description ==

With a quick activation, you can receive a daily email if you have any plugins that need updating. If no plugins have pending updates, you will not receive an email at all.  If you don't upgrade, the plugin will keep nagging you until you run the updates.
